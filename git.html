Git
Commands

Git est un outil essentiel pour le développement collaboratif de logiciels. Parmi ses principales commandes figurent git init, git clone, git add, git commit, git push, git pull, git status, git log, git branch, et git merge. Ces commandes permettent une gestion efficace des versions du code source et facilitent la collaboration entre développeurs.

Les principales commandes de Git

git init

Initialise un dépôt Git dans le répertoire courant.

Status
C:\Projects\OmegEva\commands> git init

git clone

Clone un dépôt Git distant dans le répertoire courant.

Status
C:\Projects\OmegEva\commands> git clone

git add

Ajoute des fichiers à l'index de suivi.

Status
C:\Projects\OmegEva\commands> git add

git commit

Enregistre les modifications dans le dépôt local.

Status
C:\Projects\OmegEva\commands> git commit

git push

Envoie les modifications au dépôt distant.

Status
C:\Projects\OmegEva\commands> git push

git pull

Récupère et fusionne les modifications du dépôt distant vers votre copie locale.

Status
C:\Projects\OmegEva\commands> git pull

git status

Affiche l'état des fichiers dans le répertoire de travail et de l'index.

Status
C:\Projects\OmegEva\commands> git status

git log

Affiche l'historique des validations avec les détails des auteurs et des dates.

Status
C:\Projects\OmegEva\commands> git log

git branch

Affiche la liste des branches locales et met en évidence celle sur laquelle vous êtes actuellement.

Status
C:\Projects\OmegEva\commands> git branch

git merge

Fusionne une branche spécifiée dans la branche actuelle.


Status
C:\Projects\OmegEva\commands> git merge
